# # 规则

\1. TypeScript结构化类型系统的基本规则是：如果`y`赋值给`x`，那么`y`必须包含`x`所有key

```typescript
let x = { name: "木子李" };
let y = { name: "张三丰", tel: "152288885771" }
x = y; // ok
y = x; // error
```

\2. 目标函数的参数必须在原函数中找到对应类型的参数

```typescript
let x = (a: number) => 0;
let y = (a: number, b: string) => 0;
x = y; // error
y = x; // ok
```

\3. 原函数的返回值类型必须是目标函数返回值类型的子类型。

```typescript
let x = () => ({name: "木子李"});
let y = () => ({name: "张三丰", tel: "17398888669"});
x = y; // ok
y = x; // error
```
