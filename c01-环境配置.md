# # 安装

```shell
$ npm i -g typescript
```

查看版本：

```shell
$ tsc --version
```

# # 编译

http://www.typescriptlang.org/play/index.html

## 1. tsc

```shell
$ tsc test.ts
```

## 2. vscode

1. 首先在工程首页自定义 -> 工具和语言中 选择 "Typescript" 进行安装

2. 在终端中进入项目，通过 `tsc -init` 创建 " tsconfig.json" 文件

3. 配置如下：

   ```json
   {
     "compilerOptions": {
       "target": "es5", // 编译目标平台
       "outDir": "./",    // 输出目录
     }
   }
   ```

4. 输入快捷键 `ctr + shfit + b` 选择任务进行编译
